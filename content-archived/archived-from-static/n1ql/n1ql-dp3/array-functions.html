Array functions

Array functions perform operations on arrays such as concatenate, append, prepend, remove, and length. The following table lists the array functions.

Function Description Returns
ARRAY_CONCAT(array1, array2) Concatenates the array with the input arrays. Array
ARRAY_APPEND(array, value) New array with an appended value. Array
ARRAY_PREPEND(value, array) New array with a prepended value. Array
ARRAY_REMOVE(array, value) New array with all occurrences of the value removed. Array
ARRAY_LENGTH(array) Retrieves the number of elements in the array. Integer
