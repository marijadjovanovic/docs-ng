Comparison functions

Comparison functions determines the greatest or least value from a set of values. The following table lists the comparison functions.

Function Description Returns
GREATEST(expr, expr, ...) Evaluates the expressions and finds the greatest non-NULL, non-MISSING value if the values are of the same type. Otherwise, NULL. Value or NULL
LEAST(expr, expr, ... ) Evaluates the expressions and finds the least non-NULL, non-MISSING VALUE value if the values are of the same type. Otherwise, NULL. Value or NULL
