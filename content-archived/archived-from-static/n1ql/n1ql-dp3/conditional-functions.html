Conditional functions

Conditional functions evaluate expressions to determine if the values and formulas meet the specified condition. The following tables list the conditional functions for unknowns and numbers.

Conditional functions for unknowns

The following functions are conditional functions for unknowns.

Function Description Returns
FIRSTNUM(expr1, expr2, ... ) Evaluates the all the expressions and finds the first non-NULL, non-MISSING, non-NaN, non-infinite numeric value. NULL or Integer
IFMISSING(expr, expr, ... ) Evaluates and finds the first non-MISSING value. Value
IFMISSINGORNULL(expr, expr, ... ) Evaluates and finds the first non-MISSING, non-NULL value. Value
IFNULL(expr, expr, ... ) Evaluates and finds the first non-NULL value. Value
MISSINGIF(value1, value2) If value1 equals value2 return MISSING, otherwise value1. Value
NULLIF(value1, value2 ) If valuel1 equals value2, return NULL, otherwise value1. Value1 or NULL

Conditional functions for numbers

The following functions are conditional functions for numbers.

Function Description Returns
IFINF(expr1, expr2, ... ) Evaluates and outputs the first non-infinite value (+ or -). Value
IFNAN(expr1, expr2, ... ) Evaluates and outputs the first non-NaN value. Value
IFNANORINF(expr1, expr2, ... ) Evaluates and outputs the first non-NaN, non-infinite value. Value
IFNEGINF(expr1, expr2, ... ) Evaluates and outputs the first non-infinite value. Value
IFPOSINF(expr1, expr2, ... ) Evaluates and outputs the first positive non-infinite value. Value
NANIF(value1, value2) If value1 equals value2, outputs NaN, otherwise value1. Value1 or NaN
NEGINFIF(value1, value2) If value1 equals value2, outputs negative infinity, otherwise value1. Value1 or +infinity
NEGINFIF(value1, value2) If value1 equals value2, outputs negative infinity, otherwise value1. Value1 or +infinity
POSINFIF(value1, value2) If value1 equals value2, outputs positive infinity, otherwise value1. Value1 or +infinity
