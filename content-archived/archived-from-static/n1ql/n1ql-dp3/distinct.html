DISTINCT clause

The DISTINCT clause removes duplicate result objects from the query's result set. If the DISTINCT clause is not used, the query returns all objects that meet the query conditions in a result set.

Compatibility

Compatible with Couchbase Server 2.2 and higher

Syntax

SELECT DISTINCT resultExpression

Example

The following examples show a simple DISTINCT clause using the Couchbase beer-sample bucket and N1QL tutorial.

SELECT DISTINCT * FROM beer-sample LIMIT 1               
      
SELECT DISTINCT *
  FROM tutorial
     WHERE fname = 'Ian'
