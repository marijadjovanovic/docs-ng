SELECT clause

The SELECT clause queries data from the Couchbase database.

Syntax

SELECT [ ALL | DISTINCT ] result-expr [ , ... ]

Compatibility

Compatible with Couchbase Server 2.2

Examples

The following examples are based on the N1QL tutorial.
  • Hello World!
  • System buckets
  • Jungleville
  • Jungleville piggies

Hello World!

This query displays "Hello World!".

SELECT 'Hello World!' AS Greeting

Returns:

{
  "resultset": [
    {
      "Greeting": "Hello World!"
    }
  ]
}
